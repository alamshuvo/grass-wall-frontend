## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/alamshuvo/grass-wall-frontend.git
   ```
2. Install Dependencies

```bash
cd project-name
npm install
```
3. Run the development server:
```bash 
npm run dev
```
